# John Charlie Cabilin Portfolio

My portfolio showcasing academic background, skills, projects, and contact information.

## Overview

This portfolio highlights my journey in Information Technology at Bukidnon State University, featuring projects in **C** and **HTML**, along with proficiency levels in various programming languages and tools. The site is built with clean, modern design principles and includes smooth animations for better user engagement.

## Features

- **Responsive Design**: Mobile-first approach with adaptive layouts for all screen sizes
- **Smooth Animations**: Fade-in, slide-in, and scale effects on page load and interactions
- **Interactive Cards**: Hover effects on project cards with zoom and shadow animations
- **Fixed Navigation**: Header stays visible while scrolling for easy navigation
- **About Section**: Academic information, technical & soft skills, proficiency indicators, future goals, and personal interests
- **Project Showcase**: 
  - C Language projects (BMI Calculator, Average Calculator, Temperature Converter)
  - HTML projects (Login/Sign-up Pages, Tourism Website, Portfolio)
  - Project image showcase with custom screenshot display
- **Contact Form**: Front-end form with validation and demo submission
- **Avatar**: Circular profile image with responsive sizing

## Getting Started

### Open Locally (Windows PowerShell)

```powershell
# Navigate to project folder and open in browser
ii index.html
```

Or simply drag \index.html\ to your browser.

## Customization

### Update Your Information

1. **Profile Image**: Replace the avatar image URL in the \hero\ section
2. **About Section**: Update academic info, skills, and interests
3. **Projects**: 
   - Update C project link to your GitHub repository
   - Update HTML project link to your GitHub repository
   - Replace the project showcase image URL with your screenshot
4. **Contact**: Update the email address in the contact form demo

### GitHub Links

Update the repository links in the project cards.

### Project Image

Replace the placeholder image in the Project Showcase card.

## Features Breakdown

### Navigation
- Fixed header with smooth scroll navigation
- Mobile-friendly toggle menu
- Active section highlighting

### Animations
- Hero section: Text slides in with staggered timing
- About blocks: Fade-in effects with delays
- Project cards: Scale-in animations on load
- Hover states: Lift effect, shadow glow, zoom on images

### Responsive Breakpoints
- **Mobile** (< 800px): Stacked layout, mobile nav toggle
- **Desktop** ( 1000px): Two-column About grid, side-by-side hero layout

## Notes

- The contact form is **front-end only**  it validates input but doesn't actually send emails
- Progress bars animate on page load and show skill proficiency levels
- All animations respect \prefers-reduced-motion\ for accessibility
- Avatar uses a circular crop with accent border
