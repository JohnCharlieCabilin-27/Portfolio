# John Charlie Cabilin Portfolio

My portfolio showcasing academic background, skills, projects, and contact information.

## Overview

This portfolio highlights my journey in Information Technology at Bukidnon State University, featuring projects in **C** and **HTML**, along with proficiency levels in various programming languages and tools. The site is built with clean, modern design principles and includes smooth animations for better user engagement.

## Features

- **Responsive Design**: Mobile-first approach with adaptive layouts for all screen sizes
- **Smooth Animations**: Fade-in, slide-in, and scale effects on page load and interactions
- **Interactive Cards**: Hover effects on project cards with zoom and shadow animations
- **Fixed Navigation**: Header stays visible while scrolling for easy navigation
- **About Section**: Academic information, technical & soft skills, proficiency indicators, future goals, and personal interests
- **Project Showcase**: 
  - C Language projects (BMI Calculator, Average Calculator, Temperature Converter)
  - HTML projects (Login/Sign-up Pages, Tourism Website, Portfolio)
  - Project image showcase with custom screenshot display
- **Contact Form**: Front-end form with validation and demo submission
- **Avatar**: Circular profile image with responsive sizing
